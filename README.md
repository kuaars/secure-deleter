## Безопасное удаление

приложение на Python/Tkinter, которое **перезаписывает байты файла** (один или несколько проходов), а затем удаляет файл.

### Запуск

```bash
git clone https://github.com/kuaars/secure-deleter.git
```

```bash
python secure_delete_gui.py
```

### Пункт в контекстном меню Проводника (Windows)

Чтобы добавить пункт **«Удалить с перезаписью»** в меню по ПКМ на файле:

- Запустите `install_context_menu.bat`

Чтобы убрать пункт в ПКМ:

- Запустите `uninstall_context_menu.bat`

Примечание: пункт меню запускает команду через `python`, поэтому на время работы может появляться консольное окно.

### Что делает

- Вы выбираете один или несколько файлов (или добавляете целую папку).
- Настраиваете **число проходов** и **шаблон** (random/zeros/ones/random+zeros).
- Приложение перезаписывает содержимое файла блоками, делает `fsync`, при необходимости проверяет последний проход (только для zeros/ones), опционально переименовывает файл, затем удаляет.

### Важные ограничения (прочитайте)

Схема «перезаписать и удалить» **не гарантирует** необратимость во многих реальных сценариях:

- **Снимки, бэкапы, облачная синхронизация, антивирус/карантин, индексаторы, временные файлы**: могут хранить копии.
- **Открытые/заблокированные файлы**: операция может не выполниться полностью.
